# 🎓 Student Performance AI Dashboard

## 📌 Project Description
This project is a machine learning-based system that analyzes student data and predicts student performance. It uses data preprocessing, model training, and analysis modules to build an end-to-end AI workflow.

---

## 🧠 Modules Used
- data_preprocessing.py → Cleans and prepares dataset
- model_training.py → Builds and trains machine learning model
- analysis.py → Evaluates model performance and results
- main.py → Runs the complete project workflow

---

## ⚙️ How to Run

Open terminal or command prompt and follow these steps:

### 1. Clone the repository
git clone 
https://github.com/mamathammn027/student_performancne_ai_dashboard.git

### 2. Move into project folder
cd student_performancne_ai_dashboard##


### 3. Install required libraries
pip install -r requirements.txt

### 4. Run the project
python main.py
Open your browser to the local link shown in your terminal. Network URL: http://127.0.0.1:5000
